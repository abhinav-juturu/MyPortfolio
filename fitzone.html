Fit Zone (Personalized Fitness Website)

Developed a full-featured fitness hub with exercise, nutrition, and audio integration.

Front-End Technology Stack

The platform was built focusing on user experience and cross-platform compatibility:

**HTML, CSS, JavaScript, Electron.JS**

  • **Electron.JS:** Used for packaging the web application as a cross-platform desktop application.
  • **Pure JavaScript:** Handled all interactive elements, filtering, and data management.

Key UX Features

Enabled personalized fitness experiences through several core functionalities:

  • **Exercise Filtering:** Users can sort exercises by muscle group or intensity.
  • **Food Tracking:** Basic logging capabilities for tracking nutritional intake.
  • **Motivational Audio:** Integrated music player for workouts (simulated integration).
  • **Categorized Plans:** Offering different workout and nutritional paths.

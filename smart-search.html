Overview

A prefix-based search system that mimics real-time suggestions. Queries are served with low latency using a Trie for prefix lookup and a BST to keep suggestions ranked/organized.

Highlights

  • O(L) prefix lookup via Trie; suggestions assembled in ranked order
  • Optimized memory layout and iterative traversals
  • Handles add/remove terms and frequency-based ranking

Tech Stack

  • C++17, STL
  • Trie (prefix tree), BST for ranking
  • Unit-tested modules for insert/search/delete

My Role

  • Data structure design & complexity analysis
  • Implemented traversal & ranking logic
  • Benchmarks and profiling
